Introduction¶

In this project, i will analysis TMDb movie data which is data set contains information about 10,000 movies collected from The Movie Database (TMDb), based on number of attributes like popularity, budget, revenue, runtime, genres, vote_average, release_year, budget_adj, and revenue_adj. The analysis aims to answer these questions:

  • Are the number of movies increase each year?
  • Which year has the highest / lowest release movies?
  • Is the popularity affect over years?

Conclusions¶

Finally, after I did the whole process of Data Analysis I got these result:

  • In the Data Wrangling step I take an overview of the data set see the statices numbers, check data types, and if they're missing or duplicate values, look at the distribution for attributes.
  • In the Data Cleaning step: dropped unused columns, fixed incorrect data type, and remove nulls and duplicated values to get a more accurate data set.
  • Exploratory Data Analysis step: began to ask questions about these data and try to get quick insights by doing the plotting and visualized.

In general, the industry of movies grows based on revenues per year and I discovered the runtime of the movies doesn't impact the voting average. Also, the average rate of voting is 6 and most movies have this average voting.

Limitations:¶

  • Dropped some unnecessary attributes like id, imdb_id, and etc.
  • There are duplicate rows with id 2089 and 2090, I dropped them.
  • There are a lot of null values, for example, cast, keyword, director but in the chosen attributes only in genres attribute and I dropped nulls.
  • The genres column has many values for one movie which affects the accuracy of the analysis.
  • The data type for the release_date attribute was 'object' and I converted it to 'datetime' to make sense.
